# Competitor Apps

Mac apps that export or back up an Apple Photos library to external storage. This list is research material for a future "how Photo Export compares" section on the website.

## Photos Backup Anywhere

- **Website:** <https://photosbackup.app/>
- **Platform:** macOS
- **Pricing:** Unknown (beta available via TestFlight)
- **Focus:** Full library backup with restore capability

**Key features:**
- Automatic background backups of new photos and videos
- Works with iCloud Photos "Optimize Storage" enabled
- Can restore backups back into a photo library
- Preserves non-destructive edits, albums, favorites, locations, timestamps
- Supports live photos, portraits, slow-motion, panoramas, cinematic videos, bursts

**Notes:**
- Emphasis is on backup-and-restore rather than plain file export
- Supports network storage (NAS) in addition to external drives
- Folder structure / export format not documented on the website

## Photos Takeout

- **Website:** <https://photostakeout.com/>
- **Platform:** macOS (Mac App Store, ID 1500714691)
- **Developer:** App Initio Limited
- **Pricing:** Paid (Mac App Store)
- **Focus:** Export and backup with flexible folder organization

**Key features:**
- Export by years, albums, or moments/dates
- Full-resolution photo and video export
- Metadata preservation (titles, descriptions, keywords, locations)
- Handles multi-terabyte libraries, works with "Optimize Storage"
- No 1,000-item export limit (unlike iCloud web export)
- Year folders with optional monthly sub-folders, or mirror album structure
- Choose original or edited versions
- Selective export (photos only, videos only, or both)
- Incremental exports — detects changes and syncs without re-exporting everything

**Notes:**
- Minimalist three-button UI (Years, Albums, Moments)
- Runs entirely locally, no cloud dependency
- Referenced by 9to5Mac and professional photo organizers
- Closest competitor in terms of feature overlap with Photo Export
